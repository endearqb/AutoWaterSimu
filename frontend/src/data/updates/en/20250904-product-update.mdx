# 2025-09-04/05 Update Notes

## New Features

### ASM1 Analysis System
- Added a shared analysis button system for different model types
- Implemented an ASM1-specific analysis toolkit (validation, statistics, trend analysis)
- Built the ASM1Analyzer UI with charts, tables, and summary statistics
- Integrated the analysis entry into SimulationPanel (shown only after a successful run with results)

## Interactive Water-Treatment Calculators
- AO anoxic/aerobic volume and mixed liquor recycle calculator
- Theoretical oxygen demand calculators (AOR & SOR)

## Other
- Increased “steps per hour” for ASM1 and ASM1Slim models
