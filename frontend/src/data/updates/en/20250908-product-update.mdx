# 2025-09-08 Update Notes

## New Features

### 1) Full ASM3 Integration
- Implemented the full backend integration (data models, services, API routes, migrations)
- Added access control (Ultra users and super admins only)
- Added ASM3 result rendering with 13 state variables
- Created ASM3-specific analysis components

### 2) ASM1Slim Analysis
- Built ASM1Slim-specific analysis components
- Integrated analysis actions into toolbar and simulation panel

### 3) New DWA Calculator + Performance Work
- Removed real-time calculation and added explicit “Calculate” / “Recalculate” controls
- Added calculation status indicators to avoid UI blocking
- Switched to async calculation for better interaction

## UI Improvements
- Added clear model-type badges in the toolbar (ASM1 / ASM1Slim / ASM3 / MBalance)
- Improved long-title layout and overflow handling in the toolbar
