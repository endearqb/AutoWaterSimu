# 2025-09-08 更新日志

## 🚀 新增功能

### 1. ASM3模型完整集成
* **完成ASM3后端完整集成**，包括数据模型、服务层、API路由和数据库迁移
* **实现ASM3权限控制**，仅允许ultra用户和超级管理员访问，***暂时留给自己，有需要或感兴趣可找我***
* **添加ASM3计算结果显示**，支持13个状态变量的中文标签映射
* **创建ASM3专用分析组件**
![](/assets/images/updates/0908-1.png)

### 2. ASM1Slim完整数据分析功能
* **创建ASM1Slim专用分析组件**
* **完成分析面板集成**，在工具栏和模拟面板中添加分析按钮
![](/assets/images/updates/0908-2.png)

### 3. 新增DWA计算器并性能优化
* **移除实时计算**，添加"开始计算"和"重新计算"按钮控制计算时机
* **添加计算状态指示器**，显示计算进度避免UI阻塞
* **使用异步计算**，改善用户交互体验
![](/assets/images/updates/0908-3.png)


## 🔧 界面优化改进

### 1. 工具栏流程图类型标识系统
* **动态显示模型类型Logo**，根据modelType显示不同颜色的标识
  - ASM1: 绿色 "ASM1"
  - ASM1Slim: 浅绿色 "ASM1Slim" 
  - ASM3: 蓝色 "ASM3"
  - 物料平衡: 紫色 "MBalance"
* **圆角边框设计**，与节点装饰条颜色保持一致
* **提供清晰的类型识别**，位于流程图名称前方
![](/assets/images/updates/0908-4.png)

### 2. 工具栏标题布局优化
* **修复标题过长布局问题**，长标题不再挡住锁定和折叠按钮
* **添加文本溢出处理**，长标题以省略号显示，鼠标悬停查看完整内容
* **优化布局**，确保编辑模式下输入框也不会挡住按钮
![](/assets/images/updates/0908-5.png)
