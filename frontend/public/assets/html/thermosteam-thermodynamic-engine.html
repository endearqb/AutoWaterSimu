thermosteam: BioSTEAM 的热力学引擎

一个独立、开源的 Python 库，为化学过程模拟（特别是生物精炼领域）提供强大而灵活的框架，用于处理质量与能量平衡、混合物性质估算、相平衡求解及化学反应模拟。

开源与灵活

基于 Python 原生开发，采用宽松的 NCSA 许可证，易于集成、扩展和二次开发。

专注生物精炼

为 BioSTEAM 的技术经济分析(TEA)和生命周期评价(LCA)提供坚实的热力学计算基础。

性能优化

通过 Numba JIT 编译和缓存机制，实现高效计算，支持快速迭代分析。

核心架构与依赖

`thermosteam` 构建于成熟的科学计算生态之上，采用分层和模块化的设计，确保了其功能的强大与稳定。

关键热力学模型

`thermosteam` 整合了多种热力学模型来精确求解相平衡和模拟化学反应，其中 UNIFAC 模型尤为关键。

UNIFAC 模型

基于基团贡献法预测液相活度系数，能够处理缺乏实验数据的复杂有机混合物。这使其在生物精炼领域（如醇、酸、酯混合体系）的 VLE 和 LLE 计算中具有不可替代的优势。

修正的拉乌尔定律

结合活度系数（常由UNIFAC计算）来描述中低压下非理想溶液的汽液平衡行为，是 VLE 计算的基础。

Peng-Robinson EOS

广义状态方程，目前主要用于描述纯组分的 P-V-T 性质。未来计划扩展至混合物，以处理高压或临界区的相平衡。

主要应用领域

通过与 BioSTEAM 结合，`thermosteam` 在多种生物精炼过程的模拟、设计和经济性分析中发挥了核心作用。

纤维素乙醇生产

模拟以玉米秸秆或小麦秸秆为原料，通过预处理、酶解、发酵和蒸馏等步骤生产第二代生物燃料的过程。

脂质甘蔗联产

对从特殊甘蔗中提取糖和脂质，并分别转化为乙醇和生物柴油的复杂过程进行建模与经济性评估。

生物基化学品

支持山梨酸钾、乳酸、HMF等多种高附加值生物基化学品生产路线的模拟与不确定性分析。

生态与竞品对比

`thermosteam`/BioSTEAM 在开源生态中定位明确，与商业软件相比，在灵活性、成本和特定分析领域展现出独特优势。

与功能全面的商业软件 Aspen Plus 或开源模拟器 DWSIM 不同，`thermosteam`/BioSTEAM 的核心竞争力不在于拥有最广泛的单元模型库，而在于：

  • 零成本与完全开放: 采用宽松开源许可，无任何使用费用，代码完全透明。
  • Python 原生 API: 提供极高的编程灵活性和可扩展性，便于研究人员进行定制化开发和自动化分析。
  • 聚焦 TEA/LCA 与不确定性分析: 专为新兴技术的快速经济性评估和大规模不确定性量化而设计，这是其核心优势。

这使其成为学术界和新兴技术研发领域评估创新生物过程的理想工具。
