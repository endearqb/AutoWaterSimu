水处理计算器中文预览页

本页用于快速验证 5 个 WaterTAP 风格计算器的内容结构、输入字段、结果表达和文档说明方式。 当前版本是工程快算预览，适合方案初筛、教学演示和参数敏感性体验，不用于替代 WaterTAP 正式模型、后端求解器或完整工艺设计。

本页验证什么 输入项是否合理、结果是否易懂、中文 docs 是否足够清楚、哪些计算适合继续迁入 React 前端。
本页不验证什么 不复刻 Pyomo/IDAES 严格求解，不保证与 WaterTAP 数值一一对应，不做 flowsheet 级联优化。
建议使用方式 先点“常规示例”看默认表现，再点“边界示例”观察风险提示，最后结合卡片内 Docs 判断是否值得前端集成。
