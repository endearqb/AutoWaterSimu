## 目标
- 在调色板面板底部按钮区“重置”前新增按钮“经典主题”。
- 启用后将节点切换为经典扁平风格：白底、细边框、无阴影，字体颜色沿用当前“重置”后的主题字体色；连接柄为实心小圆点，无玻璃效果。
- 保持每页面（每模型）独立主题；经典主题为 per‑model 生效，不影响其他页面。

## 关键改动
### 新增按钮
- 文件：`frontend/src/components/Flow/inspectorbar/SimulationActionPlate.tsx`
- 位置：调色板浮层底部 `HStack`（现有“重置”、“关闭”）
- 行为：点击后调用主题仓库的新方法以应用经典主题：`themeStore.applyClassicThemeForModel(modelKey)`

### 主题仓库扩展
- 文件：`frontend/src/stores/themePaletteStore.ts`
- 新增：
  - `applyClassicThemeForModel(modelKey: string)`：
    - 记录 `modelPalettes[modelKey].mode = 'classic'`（保持 per‑model 独立）
    - 设置 CSS 变量以禁用玻璃并应用经典样式：
      - `--chakra-glass-classic: '1'`
      - `--chakra-colors-glass-blur: '0px'`
      - `--chakra-colors-glass-panel: 'transparent'`（或 `'#fff'`）
      - `--chakra-colors-glass-tint-*: 'transparent'`（覆盖所有 tint：`default,input,output,asm1,asm3,asmslim,...`）
      - `--chakra-colors-glass-border: 'hsl(214.3 31.8% 91.4%)'`
      - 字体色沿用当前重置后的主题字体色：读取现有 `--chakra-colors-glass-accent-<primaryTint>`（或 `default`）并写入 `--chakra-colors-classic-text`
      - 连接柄色写入：`--chakra-colors-classic-handle: 'hsl(215.4 16.3% 46.9%)'`
  - 更新 `reset()`：
    - 清除 `--chakra-glass-classic` 标记（设为 `'0'`）
    - 恢复玻璃相关变量为默认（现有逻辑保持不变），同时移除 `--chakra-colors-classic-*`

### 玻璃样式扩展为支持经典模式
- 文件：`frontend/src/components/Flow/nodes/utils/glass.ts`
- 更新 `getGlassNodeStyles(...)` 与连接柄样式逻辑：
  - 若检测到 `document.documentElement` 上 `--chakra-glass-classic === '1'`，返回经典样式对象：
    - `backgroundColor: 'white'`
    - `border: '1px solid hsl(214.3 31.8% 91.4%)'`
    - `borderRadius: '3px'`
    - `boxShadow: 'none'`（无阴影）
    - `color: var(--chakra-colors-classic-text)`（字体颜色沿用重置后的主题）
    - 连接柄：背景 `var(--chakra-colors-classic-handle)`，边框 `1px solid white`，hover 时显示
  - 否则按现有玻璃样式分支返回
- 保持 `getGlassPanelStyles(...)` 不变（面板仍使用玻璃气泡风格）。

## 交互与一致性
- 经典主题按钮置于“重置”之前：顺序为 `[经典主题][重置][关闭]`
- 点击“经典主题”即时应用但不关闭弹层；点击“关闭”关闭弹层；弹层外点击也关闭（沿用既有互斥逻辑）。
- 每页面独立：使用现有 `modelKey` 与 `applyStoredForModel(modelKey, tintOrder)` 保持主题恢复；经典模式被存入 `modelPalettes[modelKey]`。

## 验证
- 前端类型检查：`cd frontend; npx tsc --noEmit`
- 本地预览：`cd frontend; npm run dev`，分别进入 ASM1、ASM3、ASM1Slim、物料平衡页面：
  - 切换“经典主题”后：节点白底、细边框、无阴影；字体颜色与重置后保持一致；连接柄为实心小圆点
  - 点击“重置”：回到玻璃主题（`--chakra-glass-classic` 清零，玻璃变量恢复）
  - 切换页面互不影响

## 回滚与兼容
- 所有变更均通过 CSS 变量与仓库方法控制；无侵入的节点组件改名或结构变化。
- 如需还原，仅移除按钮与仓库扩展即可；玻璃样式原逻辑保持完整。
