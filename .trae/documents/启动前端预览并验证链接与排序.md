## 目标
- 启动前端开发预览服务，提供可访问的本地 URL。
- 在预览中验证：目录侧边文件优先排序、两篇根文档位于顶部；目录页中各 Markdown 链接可正常跳转。

## 步骤
1. 在 `frontend` 目录启动开发服务：运行 `npm run dev`（Windows PowerShell 环境）。
2. 等待服务启动输出本地访问地址（如 `http://localhost:5173/`）。
3. 提供可点击的预览 URL。
4. 验证建议：
   - 侧边栏顺序：顶部应为“目录内容摘要”“废水处理技术分类总结报告”，其后先显示文件、再显示文件夹。
   - 在“目录内容摘要”页面依次点击若干条目（如“高浓度废水生物处理最佳实践_”、“芬顿工艺及其变种研究_”等），检查路由跳转是否正确。

- 不修改代码，仅启动本地服务并进行交互验证。
