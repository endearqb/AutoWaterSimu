## 目标
- 将“每小时步数”改为一个 `steps` 按钮，点击后在上方弹出气泡，提供 10、20、30、40、50、60 快捷按钮。
- 将“采样间隔”改为一个 `resample` 按钮，点击后在上方弹出气泡，提供 0.5 和 1 两个快捷按钮。

## 技术方案
- 使用 Chakra UI v3 的 Popover 组件（项目中已采用 `.Root/.Trigger/.Content` 风格）。
- `placement` 设置为 `top`（或使用 v3 的 `positioning={{ placement: 'top' }}`）以“向上弹出”。
- 触发按钮沿用现有的 `getToggleButtonStyles` 风格与强调色（`accentColor`），确保视觉统一。
- 气泡中的选项使用小号 `Button`，选中态与非选中态沿用 `getToggleButtonStyles`。
- 点击选项调用现有 `validateAndUpdateParameters`：
  - 步数：`validateAndUpdateParameters({ steps_per_hour: value })`
  - 采样：`validateAndUpdateParameters({ sampling_interval_hours: value })`

## 样式与一致性
- 触发按钮与选项按钮均沿用 `getToggleButtonStyles`，以 `accentColor` 控制边框与填充。
- `Popover.Content` 采用半透明深色背景与边框色为 `accentColor`，做轻微模糊以匹配玻璃风格；可根据需要进一步对齐 `panelStyles`。
- 保留 `config.stepsPerHour.visible` / `config.samplingIntervalHours.visible` 的显隐判断。
